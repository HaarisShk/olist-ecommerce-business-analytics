# Dataset

## Olist Brazilian E-Commerce Dataset

This project uses the **Olist Brazilian E-Commerce Public Dataset**, which contains transactional data from one of Brazil's largest online marketplaces. The dataset includes information on customers, orders, products, sellers, payments, reviews, and geolocation, making it suitable for end-to-end business analytics and dashboard development.

### Dataset Overview

- **Source:** Kaggle
- **Time Period:** September 2016 – October 2018
- **Records:** Approximately 100,000 orders
- **Domain:** E-Commerce

### Tables Used

| Table | Description |
|-------|-------------|
| customers | Customer information and location |
| orders | Order lifecycle and status |
| order_items | Products purchased in each order |
| order_payments | Payment method and transaction details |
| order_reviews | Customer review scores |
| products | Product details and categories |
| sellers | Seller information |
| geolocation | Geographic location data |
| category_translation | Portuguese to English category mapping |

### Dataset Source

https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce

### Note

The raw dataset is **not included** in this repository because it is publicly available through Kaggle. Please download it directly from the source above if you wish to reproduce this project.
